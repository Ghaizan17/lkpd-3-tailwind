media belajar

media belajar adalah sebuah website media pembelajaran simpel untuk tugas lkpd 2 tailwind dari ibu faridah.

tentang web ini

Website ini adalah sebuah media pembelajaran digital yang dibuat menggunakan Tailwind CSS di Visual Studio Code. Website ini dibuat sebagai bagian dari tugas LKPD 2 dari Ibu Faridah, sekaligus menjadi sarana belajar untuk memahami bagaimana membuat tampilan web yang sederhana, responsif, dan menarik.

kenapa disini?

Karena website ini tidak hanya sekadar tugas, tetapi juga tempat untuk belajar, berlatih, dan mengasah keterampilan dalam membangun proyek nyata dengan teknologi modern. Disini, pembelajaran dikemas agar lebih menyenangkan, mudah diakses, dan dapat menjadi inspirasi dalam pengembangan web selanjutnya.
